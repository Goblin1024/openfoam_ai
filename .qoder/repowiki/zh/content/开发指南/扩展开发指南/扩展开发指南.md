## 项目概述

OpenFOAM AI项目是一个基于大语言模型的自动化CFD仿真智能体系统，旨在通过智能化的Agent协作来简化CFD仿真流程。该项目采用模块化设计，包含多个专门的Agent来处理不同的仿真任务。

项目的核心特点包括：
- 基于AI约束宪法的硬约束验证系统
- 多Agent协作的智能体架构
- 完整的CFD仿真工作流自动化
- 强大的网格质量和物理验证能力

### 自动修复策略

自愈Agent提供了三种主要的修复策略：

1. **库朗数超限修复**：自动减小时间步长并从最新时间步重启
2. **残差爆炸修复**：减小松弛因子以提高数值稳定性
3. **残差停滞修复**：增加非正交修正器来改善收敛性

## 新Agent开发模板

### 基础Agent模板

```python
from typing import Dict, Any, Optional
from dataclasses import dataclass
from enum import Enum

class AgentState(Enum):
    IDLE = "idle"
    PROCESSING = "processing"
    COMPLETED = "completed"
    ERROR = "error"

@dataclass
class AgentResult:
    success: bool
    message: str
    data: Dict[str, Any]
    logs: List[str]

class BaseAgent:
    """Agent基类模板"""
    
    def __init__(self, config: Optional[Dict[str, Any]] = None):
        self.config = config or {}
        self.state = AgentState.IDLE
        self.history: List[Dict[str, Any]] = []
    
    def process_input(self, input_data: Any) -> AgentResult:
        """处理输入数据"""
        try:
            self.state = AgentState.PROCESSING
            result = self._execute_task(input_data)
            self.state = AgentState.COMPLETED
            return AgentResult(
                success=True,
                message="处理成功",
                data=result,
                logs=self.history
            )
        except Exception as e:
            self.state = AgentState.ERROR
            return AgentResult(
                success=False,
                message=str(e),
                data={},
                logs=self.history
            )
    
    def _execute_task(self, input_data: Any) -> Dict[str, Any]:
        """执行具体任务 - 需要子类实现"""
        raise NotImplementedError
    
    def validate_config(self, config: Dict[str, Any]) -> Tuple[bool, List[str]]:
        """验证配置 - 可选实现"""
        return True, []
    
    def get_status(self) -> Dict[str, Any]:
        """获取Agent状态"""
        return {
            "state": self.state.value,
            "config": self.config,
            "history_length": len(self.history)
        }
```
